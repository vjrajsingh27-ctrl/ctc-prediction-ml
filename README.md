# ctc-prediction-ml
# Predictive Modeling of CTC Using Machine Learning

## Overview
This project builds a regression model to predict CTC (salary) based on a dataset of 1,000+ employee records, using a systematic, experiment-driven approach to find the most reliable model.

## Approach
- Cleaned, engineered, and preprocessed a dataset of 1,000+ records, including feature scaling and categorical encoding
- Tested 5 regression models across 4 configurations each to compare performance
- Tuned hyperparameters to optimize the best-performing model

## Results
- **Best model:** Random Forest Regressor
- **Performance:** R² = 0.6955
- Achieved a 20% improvement in prediction accuracy through feature scaling, encoding, and hyperparameter tuning
